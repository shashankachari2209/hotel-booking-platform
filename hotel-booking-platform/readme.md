# Hotel Booking Platform

A web-based hotel reservation system where users can search hotels, check room availability, and book rooms online.

## Features
- User registration and login
- Hotel and room listing
- Room availability checking
- Online booking system
- Booking history tracking

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Setup Instructions

1. Install XAMPP
2. Place project folder inside htdocs
3. Import database from database/hotel_booking.sql
4. Start Apache and MySQL
5. Open http://localhost/hotel-booking-platform
